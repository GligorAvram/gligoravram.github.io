A program to help calculate the conversion rates between a number of foreign currencies

I made this program to experiment with the Java UI tools. I actually used them in the past as well, but I felt like I needed a refresher on them.

The program was made using only Java. It features an user interface built in Java FX and uses an API to obtain the values of the currencies it's set to calculate the exchange values for.
