The DNS report maker

An easy way to get the DNS information you need to configure your email system

Click the icon below to visit the repository of this program

An application that will get the MX, DMARC, DKIM and SPF records of a domain. It will also dive deep into the SPF record to break it down into all of its lookups and IPs

I built this application for work because one of our competitors has a similar feature, although IMO broken, and in order to simplify the initial information gathering phase of a project

The program uses the DNSJava DNS querying library in order to make the DNS lookups. The most chalenging part of making this project was breaking down the SPF. I used a custom made tree with a variable number of children per branch that also needs to keep track of the non-lookup information at the same level. Traversing it was done using a depth first pre-order traversal.
