The Manabi Japanese learning website

Probably the only website about learning Japanese in Romanian

Click the icon below to visit the repository of this program

I started learning Japanese a few years ago because I found I found it to be a fascinating language, and upon discovering that the only websites in Romanian about this language were only created to advertise clubs, groups and classes, or only provide very limited knowledge about this language, I decided to make a website about learning it dedicated to the serious learner, not the language dabbler.

The website is set to launch in the near future, if it wasn't launched already. Currently I'm still gathering some resources for it related to learning the language, but the programming section for it is done for the time being.

The website is hosted on my personal Linux server, using Jenkins and Tomcat to manage the deployment. It features a blog section and a users system linked to a MySQL database, it allows downloading files from the server, CSS and Javascript logic to handle the frontend display of Japanese characters on mouse-over with Ruby text etc. In the future I'm considering allowing the creation of regular users (currently only one user is created in the database with Admin rights) and the creation of a Forum feature, depending on the success of the website.
