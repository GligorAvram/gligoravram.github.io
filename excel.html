TheExcel Parser

For when you need to grade your Email Service Provider

Click the icon below to visit the repository of this program

A program meant to read an Excel file and create another one grading the difficulty of setting up your email senders

I started developing this application when a new workflow was proposed at work, where we set up the email senders of a domain based on their difficulty to set up. The flow invoved grading services in a terminal with no option to save the progress or undo mistakes, which quickly became frustrating.

The program uses Spring and the Apache POI library to parse the actual XLSX files. The programing section is almost complete, only needing the grade value variables, which I was not provided with. The workflow proved to be unfavorable with both agents and clients and we reverted back to setting them up based on the volume of emails sent, thus causing me to drop the project.
