Main technologies I work with

Java

My go to language when it comes to programming. I like that it's very friendly, type-safe and has a plenthora of API you can work with (among other things). Also, its "Write once, run everywhere" approach is a time saver for running applications both on Windows, Linux and Mac OS X.

Spring

The Spring framework automates and simplifies a large part of the tedious aspects of making an application, allowing me to concentrate on the unique aspects of the application I'm developing. I highly recommend it for anybody who wants to make WebApps.

Javascript, HTML, CSS

The indispensable trio of web programming. Even though my visual design skills are not as good as I'd like them to be, CSS and especially Javascript allow enriching the visual aesthetics and user functionality of the applications I'm building.
