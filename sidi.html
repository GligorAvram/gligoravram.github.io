A trip indexing program

I created this project in order to take the final exam for the Informal School of Informatics Java Development course.

The project allows the creation of users, indexing trips and adding pictures to these trips that can be viewed in the browser afterwards. The program was built using Spring, MySQL, HTML and CSS. I learned a lot by building this application however I do not plan to return to it to add any new features of improve its functionality.
